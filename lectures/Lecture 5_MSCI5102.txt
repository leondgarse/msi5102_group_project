Hamming Distance
▪ The Hamming distance between two strings of the same length
equals the number of positions in which two strings have different letters
▪ Examples:
-- H(“ACTGA”, “ATTGG”) = 2
-- H(“00110”, “01011”) = 3

Hamming distance for categorical feature vectors
▪ The Hamming distance between two data points with
categorical features is the number of features in which their values differ.
▪ Example:
Person A: (Red, Small, Male, Urban)
Person B: (Blue, Small, Female, Urban)
The Hamming distance between A and B is 2.

Idea of K-means Clustering
▪ K-means algorithm repeatedly partitions data points into
a pre-specified number (K) of non-overlapping clusters
▪ K is a hyperparameter
▪ Each cluster is represented by its centroid,
i.e., the arithmetic mean of the points in the cluster
▪ K-means minimizes the within-cluster variance

K-Means Pros and Cons
Pros
▪ Simple!
▪ K-means is fast and works well on very large datasets
▪ Can help visualize the data and facilitate detecting trends or outliers
▪ The k-means algorithm is among the most used algorithms
Cons
▪ The hyperparameter K must be decided before the algorithm starts
▪ The final assignment of data points to clusters can depend on the initial
location of the centroids, as well as K

• Assignment: For each data point:
-- Compute Hamming distance to each center
-- Assign the point to the cluster with the min. Hamming distance
• Update: For each cluster:
-- Update the cluster center by computing the mode

Hierarchical Clustering Method

▪ K-Means
– Divides objects into non-overlapping clusters such that
each data object is in exactly one cluster
▪ Hierarchical clustering
– Output a set of nested clusters organized using a dendrogram
– It does not require specifying the number of clusters in advance
– It allows the analyst to examine various partitions of data with
different granularity before deciding which one to use.
– Most widely used method for gene expression data, enabling visualization

Case Studies of Gene Expression Clustering Analysis

▪ Cancer is a mutation disease
For example, BRCA1 and BRCA2 are two mutational genes causing hereditary breast cancer
▪ Clinical (IHC/FISH) subtypes
-- Hormone-receptor positive (HR+, usually ER+, PR+):
the most common; tends to grow more slowly and is treated with endocrine therapy
-- HER2-positive (HER2+ with/without HR):
more aggressive but highly targetable with anti-HER2 agents
-- Triple-negative (ER–/PR–/HER2–):
Lacks targets for endocrine or HER2 therapy; managed with chemotherapy
▪

Molecular “intrinsic” subtypes (PAM50).
-- Luminal A (ER+, low proliferation, best prognosis)
-- Luminal B(ER+, higher proliferation, worse prognosis than LumA),
-- HER2-enriched (often HER2 pathway active)
-- Basal-like (often overlaps with triple-negative)
-- Normal-like (less commonly used clinically)
